## 📌 Problem Statement

The objective of this project is to clean and preprocess the Titanic dataset by handling missing values, removing duplicate records, renaming columns, and preparing the dataset for further analysis.


## 🎯 Objective

- Handle missing values
- Remove duplicate records
- Rename columns
- Prepare a clean dataset for analysis


## 📂 Dataset

- Dataset: Titanic Dataset
- Source: Kaggle
- File Used: `titanic.csv.csv`


## 🛠️ Tools & Technologies

- Python
- Pandas
- Google Colab / Jupyter Notebook


## 🔄 Data Cleaning Steps

1. Imported the dataset using Pandas.
2. Checked dataset information and structure.
3. Identified missing values.
4. Filled missing values in the **Age** column using the median.
5. Filled missing values in the **Embarked** column using the mode.
6. Removed duplicate records.
7. Removed the **Cabin** column due to a large number of missing values.
8. Renamed important columns for better readability.
9. Saved the cleaned dataset.


## 📊 Output

- Cleaned Titanic Dataset
- Ready for Exploratory Data Analysis (EDA)

Output File:

`cleaned_titanic.csv`
